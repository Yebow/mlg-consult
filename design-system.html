LANDING

Glow Spotlight Card

Utilisé dans le hero parallax, les service cards et la section squad. Le glow suit le pointeur via 4 CSS custom properties injectées par JS.

Strategy
Think
Think.
Before you build the wrong thing.
Start a sprint

Le spotlight radial suit les coordonnées --x / --y du curseur, injectées globalement sur tous les [data-glow] éléments par un seul listener.

Deux pseudo-éléments ::before / ::after animent la bordure lumineuse. Le tout est désactivé sur mobile (no pointer) via touch-action: none.

<!-- JS global (une seule fois par page) -->
document.addEventListener('pointermove', e => {
  document.querySelectorAll('[data-glow]').forEach(c => {
    c.style.setProperty('--x', e.clientX);
    c.style.setProperty('--xp', (e.clientX / window.innerWidth).toFixed(4));
    c.style.setProperty('--y', e.clientY);
  });
});

<!-- HTML -->
<div class="pc-card" data-glow>
  <div data-glow-inner></div>
  ...contenu...
</div>
